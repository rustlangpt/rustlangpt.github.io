Instalação do Rust: Guia Completo para Linux, Windows e macOS

Guia detalhado para instalar o Rust em diferentes sistemas operativos usando rustup, baseado na documentação oficial

Introdução

A instalação do Rust é surpreendentemente simples e uniforme em todas as plataformas principais. A linguagem utiliza uma ferramenta chamada rustup que funciona como um instalador e gestor de versões oficial. O rustup é uma ferramenta de linha de comando para gerir versões do Rust e ferramentas associadas, conforme documentado no The Rust Programming Language Book.

Neste artigo, vamos explorar o processo de instalação do Rust em diferentes sistemas operativos, baseando-nos exclusivamente na documentação oficial para garantir que está sempre atualizado com as melhores práticas.

O que é o Rustup?

O Rustup instala os binários da linguagem Rust a partir dos canais de lançamento oficiais, permitindo alternar facilmente entre compiladores stable, beta e nightly e mantê-los atualizados. Esta ferramenta:

  • Gere múltiplas versões do compilador Rust
  • Facilita cross-compilation com builds binários da biblioteca standard para plataformas comuns
  • Funciona em todas as plataformas que o Rust suporta, incluindo Windows
  • Mantém atualizações automáticas das toolchains

Para informação detalhada sobre o rustup, consulta The Rustup Book.

Instalação no Linux e macOS

Pré-requisitos

Antes de instalar o Rust em sistemas Unix-like, é importante garantir que tem as dependências necessárias:

Para Linux: Os utilizadores de Linux devem geralmente instalar GCC ou Clang, de acordo com a documentação da sua distribuição. Por exemplo:

# Ubuntu/Debian
$ sudo apt update
$ sudo apt install build-essential

# Fedora
$ sudo dnf groupinstall "Development Tools"
$ sudo dnf install gcc

# Arch Linux
$ sudo pacman -S base-devel

# openSUSE
$ sudo zypper install -t pattern devel_basis

Para macOS: No macOS, pode obter um compilador C executando: xcode-select --install

$ xcode-select --install

Processo de Instalação

Se estiver a usar Linux ou macOS, abra um terminal e digite o seguinte comando:

$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

O que este comando faz:

  • --proto '=https': Força HTTPS para segurança
  • --tlsv1.2: Utiliza TLS versão 1.2 ou superior
  • -sSf: Flags do curl para operação silenciosa com falha em erros HTTP

Configuração do PATH

Após a instalação, o rustup instala rustc, cargo, rustup e outras ferramentas padrão no diretório bin do Cargo. No Unix está localizado em $HOME/.cargo/bin.

Para ativar o Rust na sessão atual:

$ source ~/.cargo/env

Ou simplesmente reinicie o terminal. O script de instalação automaticamente adiciona o PATH ao seu ficheiro de perfil shell (.bashrc, .zshrc, etc.).

Verificação da Instalação

Para verificar se tem o Rust instalado corretamente, abra uma shell e digite esta linha: rustc --version

$ rustc --version
rustc 1.75.0 (82e1608df 2023-12-21)

$ cargo --version
cargo 1.75.0 (1d8b05cdd 2023-11-20)

Instalação no Windows

Métodos de Instalação

O Windows oferece duas formas principais de instalar o Rust:

  1. Método Recomendado: Usando o instalador visual
  2. Método Alternativo: Via Windows Subsystem for Linux (WSL)

Consulta informação adicional aqui.

Método 1: Instalador Visual (Recomendado)

No Windows, vá a https://www.rust-lang.org/tools/install e siga as instruções para instalar o Rust.

Passos detalhados:

  1. Descarregue o instalador:

    • Visite https://www.rust-lang.org/tools/install
    • Para instalar o Rust, descarregue e execute rustup-init.exe

  2. Execute o instalador:

    • Execute rustup-init.exe como administrador (recomendado)
    • Siga as instruções no ecrã

  3. Instale as ferramentas de desenvolvimento: Em algum momento da instalação, será solicitado que instale o Visual Studio. Isto fornece um linker e as bibliotecas nativas necessárias para compilar programas.

Visual Studio Build Tools

Isto requererá Visual C++ Build Tools 2019 ou equivalente. Pode instalar:

Opção 1 - Visual Studio Installer:

  1. Descarregue Visual Studio Build Tools
  2. Execute o instalador
  3. Selecione "C++ build tools"
  4. Instale os componentes necessários

Opção 2 - Visual Studio Community (Gratuito):

  • Inclui todas as ferramentas necessárias
  • IDE completa se desejar desenvolver em Windows

Método 2: Windows Subsystem for Linux (WSL)

Se for utilizador do Windows Subsystem for Linux, execute o seguinte no seu terminal, depois siga as instruções no ecrã para instalar o Rust:

$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh

Vantagens do WSL:

  • Ambiente Linux completo
  • Ferramentas Unix nativas
  • Melhor integração com ecosistemas multiplataforma

Verificação no Windows

Após a instalação, verifique usando Command Prompt ou PowerShell:

Command Prompt:

> rustc --version
> cargo --version

PowerShell:

PS> rustc --version
PS> cargo --version

Configuração do PATH no Windows

Se não vir esta informação, verifique se o Rust está na sua variável de sistema %PATH%:

Command Prompt:

> echo %PATH%

PowerShell:

PS> echo $env:Path

O rustup automaticamente adiciona %USERPROFILE%\.cargo\bin ao PATH.

Gestão de Versões com Rustup

Atualizações

Uma vez que o Rust está instalado via rustup, atualizar para uma versão recém-lançada é fácil. A partir da sua shell, execute o seguinte script de atualização: rustup update

$ rustup update

Instalação de Toolchains Específicas

O rustup permite instalar diferentes versões do Rust:

# Instalar a versão mais recente stable
$ rustup install stable

# Instalar a versão beta
$ rustup install beta

# Instalar a versão nightly
$ rustup install nightly

# Instalar uma versão específica
$ rustup install 1.75.0

Alternar Entre Versões

# Definir versão padrão
$ rustup default stable

# Usar versão específica para um projeto
$ rustup override set nightly

Listar Toolchains Instaladas

$ rustup toolchain list
stable-x86_64-unknown-linux-gnu (default)
beta-x86_64-unknown-linux-gnu
nightly-x86_64-unknown-linux-gnu

Documentação Local

A instalação do Rust também inclui uma cópia local da documentação para que possa lê-la offline. Execute rustup doc para abrir a documentação local no seu browser.

$ rustup doc
$ rustup doc --book      # Abre "The Rust Programming Language"
$ rustup doc --std       # Abre documentação da biblioteca standard
$ rustup doc --reference # Abre "The Rust Reference"

Desinstalação

Para desinstalar o Rust e rustup, execute o seguinte script de desinstalação a partir da sua shell: rustup self uninstall

$ rustup self uninstall

O que remove:

  • Todas as toolchains instaladas
  • Rustup em si
  • Modificações no PATH
  • Documentação local

Configuração de Editors e IDEs

Visual Studio Code

Para uma experiência de desenvolvimento completa, recomenda-se:

  1. Extensão rust-analyzer:

    • Análise de código em tempo real
    • Auto-completion inteligente
    • Navegação de código avançada

  2. Extensão CodeLLDB:

    • Debugging completo
    • Breakpoints e inspeção de variáveis

Outros Editores Populares

  • IntelliJ IDEA: Plugin IntelliJ Rust
  • Vim/Neovim: rust.vim plugin
  • Emacs: rust-mode
  • Sublime Text: Rust Enhanced

Para uma lista atualizada, consulte https://www.rust-lang.org/tools.

Verificação Final da Instalação

Execute estes comandos para confirmar que tudo está funcionando:

# Verificar versões
$ rustc --version
$ cargo --version
$ rustup --version

# Criar e testar projeto simples
$ cargo new hello_rust
$ cd hello_rust
$ cargo run

Se ver "Hello, world!" como output, a instalação foi bem-sucedida!

Conclusão

A instalação do Rust é um processo straightforward e bem documentado, graças ao rustup. As garantias de estabilidade do Rust asseguram que todos os exemplos do livro que compilam continuarão a compilar com versões mais recentes do Rust, tornando a linguagem uma escolha sólida para projetos de longo prazo.

O rustup não é apenas um instalador, mas uma ferramenta completa de gestão de toolchains que torna fácil trabalhar com diferentes versões do Rust, cross-compilation, e manter tudo atualizado. Com uma instalação correta, está pronto para começar a sua jornada com uma das linguagens de programação mais inovadoras e seguras disponíveis hoje.

Lembre-se de consultar sempre a documentação oficial para informações mais detalhadas e atualizadas sobre qualquer aspecto do Rust que desejar explorar.
